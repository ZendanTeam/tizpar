تیزپر (Tizpar)

Swift Persian Developer Toolkit — ابزارک توسعه‌دهنده فارسی
v0.0.1 Python 3.10+ MIT License

📋 معرفی

تیزپر یک ابزارک توسعه‌دهنده فارسی است که ابزارهای مفید و کاربردی را برای توسعه‌دهندگان ایرانی و فارسی زبان فراهم می‌کند. با تیزپر می‌توانید به راحتی اعداد را تبدیل کنید، متن فارسی را تشخیص دهید، اطلاعات سیستم را مشاهده کنید و کلی کار مفید دیگر انجام دهید.

📝 فارسی

تبدیل اعداد، تشخیص متن فارسی، حذف اعراب

tizpar persian to-persian 123

💻 سیستم

اطلاعات سیستم، CPU، حافظه، دیسک

tizpar system info

🌐 شبکه

IP عمومی، بررسی URL، پینگ

tizpar network ip

🔄 تبدیل

JSON↔YAML، Base64

tizpar convert json2yaml '{}'

📦 نصب

pip install git+https://github.com/ZendanTeam/tizpar.git

یا از روی سورس:

git clone https://github.com/ZendanTeam/tizpar.git
cd tizpar
pip install -e .

🚀 شروع سریع

# نمایش نسخه
tizpar --version

# تبدیل اعداد
tizpar persian to-persian "Hello 123"
# خروجی: Hello ۱۲۳

# بررسی سیستم
tizpar system info

# مشاهده IP
tizpar network ip

🧪 تست

pip install pytest pytest-cov
pytest -v --cov=src/tizpar
